<h1 align="center">Lucene 模板</h1>

## 技术栈

* `jdk8`、`maven3.8.1`
* `SpringBoot2.6.4`、`Lucene6.6.0`、`Sqlite`

## 如何启动

1. 把项目拉取下来
2. 用`idea`打开，自动编译下载完依赖后直接启动即可；无需修改任何配置

## 使用

打开链接：http://localhost:8080/，开始访问

输入关键字 “人参”即可进行全文检索了
